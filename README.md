## Install
Clone this repo.

```bash
git clone https://github.com/wick3dr0se/fm;cd fm
```

## Usage
```bash
$ fm
```

if you haven't installed fm, you can execute it as a script.

```bash
$ bash fm.sh
```

## Interface Controls
Q   ...   Quit  
D   ...   Create directory  
F   ...   Create file  
X   ...   Delete marked file/directory

### Movements
←, H   ...   Back a directory  
↓, J   ...   Move down  
↑, K   ...   Move up  
→, L   ...   Enter file/directory
